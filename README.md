# Grocery-app
### About the Project -
This is a Grocery Store Management Project which allows the admin (Shopkeeper) to modify the products of the shop and enter each order details.

## Built With

- Frontend: HTML/CSS/JS
- Backend: Python, Flask
- Database: MySQl
